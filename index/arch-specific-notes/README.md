<b>注意</b>： 与其他发行版一起工作的人可能会发现 Gentoo 对体系结构支持的观点有些陌生。我们不遵循 Debian 尝试提供特定版本的标准软件包并将其安装到每个平台上的尝试，而是尝试提供在任何情况下都能最好地发挥作用的任何东西。
